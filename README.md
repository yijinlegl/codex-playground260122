# Neon Pulse Lab · 未来科技感 Landing Page

一个纯前端、可直接打开 `index.html` 的作品展示级动态 landing page，面向 AI / 未来科技产品发布与作品集展示。

## 亮点功能

- **深色霓虹视觉**：玻璃拟态卡片、发光描边、霓虹渐变主色。
- **动态 Canvas 背景**：星云粒子与能量连线，帧率稳定可控。
- **3D 科技交互体**：原生 Canvas 绘制能量核心，支持鼠标旋转与点击能量爆发。
- **滚动驱动动画**：卡片淡入上浮、数据计数增长。
- **鼠标交互**：卡片高光跟随、整体轻微视差。
- **主题切换 + 低性能模式**：三种科技主题可切换，支持自动性能降级。

## 交互方式

- **主题切换**：右上角三枚主题按钮（星云 / 量子 / 极光）。
- **低性能模式**：开关可关闭复杂动效，首次加载会根据设备性能自动降级。
- **能量核心**：在首屏右侧，鼠标移动可旋转，点击触发能量爆发与波纹扩散。

## 性能模式说明

- 低性能模式会降低粒子数量并关闭复杂连线与部分动态旋转。
- 若设备内存较低、核心数较少或开启了减少动效偏好，页面会自动进入低性能模式。

## 直接部署到 GitHub Pages

1. 将本仓库推送至 GitHub。
2. 进入仓库 **Settings → Pages**。
3. 选择 **Deploy from a branch**。
4. 选择当前分支与根目录（`/root`）。
5. 保存后即可通过 GitHub Pages 访问该 landing page。

## 项目结构

- `index.html`：页面结构与内容。
- `styles.css`：主题与视觉样式。
- `script.js`：Canvas 动画、交互与性能控制。
- `README.md`：项目说明。
